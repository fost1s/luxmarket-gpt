Ссылка на оплату ChatGPT — старый Stripe

Самый простой способ — перетащите кнопку в закладки и нажмите её на chatgpt.com.

Вариант 1 — кнопка‑закладка (рекомендуется)
Совет: Зажмите ЛКМ и перетащите кнопку ниже на панель закладок. Если не получается — клик правой кнопкой (ПКМ) → «Добавить в закладки». На chatgpt.com просто нажмите эту закладку.
Получить ссылку Stripe Открыть chatgpt.com

После клика по закладке появится окно со ссылкой вида https://pay.openai.com/c/pay/... — скопируйте её.

Вариант 2 — альтернативный способ (через консоль)
  1. 1. Зайдите на сайт https://chatgpt.com и войдите в свой аккаунт.
  2. 2. Нажмите F12 (или Ctrl+Shift+I) — откроется окно разработчика.
  3. 3. Перейдите во вкладку Console.
  4. 4. Напишите allow pasting → Enter, затем раскройте код ниже, скопируйте и вставьте в консоль.
    javascript
    const a = { plan_name: 'chatgptplusplan', billing_details: { country: 'US', currency: 'USD' }, promo_code: null, checkout_ui_mode: 'redirect', }; const authReq = await fetch('/api/auth/session', { credentials: 'include' }); const authRes = await authReq.json(); const authToken = authRes.accessToken; const res = await fetch('https://chatgpt.com/backend-api/payments/checkout', { body: JSON.stringify(a), method: 'POST', credentials: 'include', headers: { 'Content-Type': 'application/json', authorization: `Bearer ${authToken}`, }, }); const data = await res.json(); data.url;
  5. 5. Нажмите Enter — в консоли появится ссылка, начинающаяся с https://pay.openai.com/c/pay/...
  6. 6. Скопируйте эту ссылку и отправьте её нам.
